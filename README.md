Rock Paper Scissors Game 🎮

A fun Rock-Paper-Scissors game built with React and TailwindCSS, featuring emoji shuffle animations for a playful experience.

Features
Choose between Rock ✊, Paper ✋, or Scissors ✌️.
Computer choice is randomly selected.
Both user and computer emojis shuffle briefly before showing the final choice.
Keeps track of scores for user and computer.
Reset button to start a new game.
Fully responsive and styled with TailwindCSS.

Installation
Clone the repository:
git clone <your-repo-url>
cd rock-paper-scissors
Install frontend dependencies:
cd frontend
npm install
Start the development server:
npm run dev
Open your browser at http://localhost:5173
 (default for Vite).

How to Play
Click Rock, Paper, or Scissors.
Emojis shuffle briefly to show choices.
The winner is displayed below along with the scoreboard.
Click Reset Game to start over.
Technologies Used
React – Frontend framework
TailwindCSS – Styling
Vite – Dev server and build tool
